# Birlik Oto CRM - Automotive Service Management

A full-stack, real-world internal tool developed for **Birlik Oto** to streamline automotive service and car wash operations. This project serves as a digital brain for the business, managing customer relationships and daily service queues.

- **Real-time Service Queue:** Live tracking of vehicles in "Waiting", "In Progress", and "Completed" states.
- **Smart License Plate Formatting:** Localized Turkish license plate validation and auto-formatting.
- **Customer Directory (Fihrist):** Searchable database of customers with integrated service history.
- **Interactive History Logs:** Deep-dive into specific vehicle records to track previous maintenance and car wash frequency.
- **Responsive Design:** Dark-mode UI optimized for desktop and mobile shop-floor usage.
